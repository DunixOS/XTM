# XTM

### Xbox Token Manager

XTM (Xbox Token Manager) is a desktop application for managing Xbox authentication
tokens and performing authenticated Xbox account actions through a modern graphical
interface. It is designed to simplify working with multiple tokens by providing a
centralized panel for token management and supported Xbox Live actions.

Unlike command-line utilities, XTM provides an intuitive GUI while remaining fast,
lightweight, and suitable for both individual and large collections of authentication
tokens.

---

## Token Requirements

XTM requires valid Xbox authentication tokens.

Supported tokens:

- XBL3.0 authentication format
- Existing valid Xbox Live authentication tokens
- Tokens with the required permissions for the actions being performed

Authentication tokens:

- Have a limited lifespan
- Must be renewed after expiration
- Are never generated by XTM
- Remain entirely under your control
